git

Very basic question to help keep my repo clean.

Installation

clustergit seems a good candidate

cd ~/Applications
git clone git@github.com:mnagel/clustergit.git
# add export PATH="$PATH:$HOME/Applications/clustergit" to ~.bashrc
source ~.bashrc
